
Khẩu lệnh: ghi "updoc"

Lệnh làm gì:
- giúp update thông tin của new feature vào tài liệu prd.md file
- giúp ghi nhận lại thông tin các thay đổi vào các file be-all-structure.md
- nếu là chức năng mới, thì tạo ra file structure tương ứng rồi ghi lại thông tin về các file đã làm
- cập nhật vào user story các mục đã hoàn thành



cách làm việc trên từng file:
- be-all-structure.md, giúp hiểu overview vè toàn bộ file trong hệ thống mà không cần scan
- fea-{feature}-structured.md, chứa chi tiết về thông tin và nhiệm vụ của từng file trong 1 feature cụ thể, hoặc các additional note liên quan tới feature đó.
- prd.md, giúp giới thiệu tổng quan về feature



Check list:
- prd file đã được cập nhật chưa
- nếu là new feature thì đã có file structure trong folder "structures" chưa
- đã cập nhật thông tin vào file be-all-structure.md
- user story đã được check các mục đã hoàn thành chưa
